# CourseRegistrationSystem

## 基本架构
0. 采用前后分离的开发方式，Restful的接口设计风格
1. SpringBoot+Mybatis作为后端
2. 前段采用layuimini作为开发框架
3. 使用fastmybatis作为Mybatis代码生成器
4. navicat作为数据库查看软件
